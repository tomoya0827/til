# 制御構文

・if命令  
if命令は、与えられた条件式がtrue/false のいずれかであるによって、実行すべき処理を分岐する命令。

    <?php
    $x = 10;
    if ($x === 10) {
      print '変数$xは10です。';
    } else {
      print '変数$xは10ではありません。';
    }

  ・if命令-多岐分岐  
  
      <?php
    $x = 35;
    if ($x > 20) {
      print '変数$xは20より大きいです。';
    } elseif ($x > 10) {
      print '変数$xは10より大きく、20以下です。';
    } else {
      print '変数$xは10以下です。';
    }

elseifブロックを利用する場合には、条件式を範囲の狭いものから順に記述する。
